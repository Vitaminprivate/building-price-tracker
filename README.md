# Building Price Tracker

## Project Documentation

Building Price Tracker is a tool designed to help users track the prices of buildings over time. It allows users to input various buildings, their prices, and any changes that occur, providing analysis and reports on price trends.

### Features
- **Price Tracking**: Keep track of price changes for multiple buildings.
- **Trend Analysis**: Analyze historical price trends to make informed decisions.
- **User-Friendly Interface**: Easy-to-use interface to input and access data.

## Usage Instructions

1. **Installation**: Clone the repository:
   ```bash
   git clone https://github.com/Vitaminprivate/building-price-tracker.git
   cd building-price-tracker
   ```
2. **Run the Application**: Use the command below to start the application:
   ```bash
   python main.py
   ```
3. **Input Building Data**: Follow the prompts to enter building names and prices.
4. **View Reports**: Access the analysis reports through the menu options.
